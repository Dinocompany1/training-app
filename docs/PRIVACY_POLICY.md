# Privacy Policy (Template)

Last updated: 2026-02-18

This app helps users plan and track workouts.

## Data We Process
- Workout data entered by the user (sessions, exercises, notes).
- Optional profile data entered by the user.
- Optional body-progress photos selected by the user.

## How Data Is Used
- To provide workout planning, tracking, and progress features.
- To keep user data available between app launches.
- Optional cloud sync if configured.

## Third-Party Services
- Supabase (optional, for sync)
- OpenAI-compatible backend endpoint (optional, for AI coach responses)

## Data Sharing
- We do not sell user data.
- Data is only shared with configured infrastructure required to provide the feature.

## Data Retention
- Data is kept until the user removes it or uninstalls the app.

## User Rights
- Users can edit or delete workout entries at any time.
- Users can request deletion if backend account-based storage is enabled.
